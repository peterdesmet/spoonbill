# Eurasian Spoonbill

## Introduction

Eurasian Spoonbill is a theme for [Pelican](http://docs.getpelican.com/) powered blogs, forked from [Spoonbill](https://github.com/peterdesmet/spoonbill). We adapt and use it for our project blog.
